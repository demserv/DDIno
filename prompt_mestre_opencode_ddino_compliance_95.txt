  - 480x320
  - DC = GPIO16
  - RST = GPIO21
  - BL = GPIO47

  - XPT2046
  - CS = GPIO38
  - IRQ = GPIO39

  - PZEM-004T v4.0
  - UART TX = GPIO17
  - UART RX = GPIO18
  - 9600 8N1
- DS18B20:
  - DQ = GPIO4
- Relés:
  - P01 = GPIO5
  - P02 = GPIO6

Arquivo: firmware/drivers/driver_relay.c
Implemente regras:
- P01 → PIN_RELAY_P01_GPIO.
- P02 → PIN_RELAY_P02_GPIO.
- P03..P10 → MCP23017 GPA0..GPA7.
- Active-high.
- Estado interno s_relay_state uint16_t.
- relay_init_safe coloca todos OFF antes de qualquer outra operação.
- relay_logical_on(P01) deve verificar se P02 está ligado quando roles forem heater/cooler; se não houver role ainda, implemente hook para plug_manager validar.
- relay_all_off deve tentar desligar P01/P02 via GPIO e P03..P10 via MCP; se MCP falhar, registrar erro e retornar falha, mas manter GPIOs seguros.

Exemplo:
```c
static esp_err_t relay_set_gpio(uint8_t plug_id, bool on)
{
    int gpio = -1;
    if (plug_id == 1) gpio = PIN_RELAY_P01_GPIO;
    else if (plug_id == 2) gpio = PIN_RELAY_P02_GPIO;
    else return ESP_ERR_INVALID_ARG;

    return gpio_set_level(gpio, on ? HW_RELAY_ACTIVE_LEVEL : HW_RELAY_SAFE_OFF_LEVEL);
}
```

Critérios de aceite:
- Todos relés OFF após relay_init_safe.
- Todas as cargas OFF em SAFE_OFF/EMERGENCY.
- Nenhum serviço usa gpio_set_level para relés.
