# Regras Elétricas

- RB-ELET-001: Potência total ≤ 1500W.
- RB-ELET-002: Corrente por plug ≤ 10A.
- RB-ELET-003: Sobrecarga → SAFE_OFF após tempo configurável.
- RB-ELET-004: Curto → SAFE_OFF imediato.
- RB-ELET-005: Subtensão (<100V) → SAFE_OFF após 5s.
- RB-ELET-006: Sobretensão (>140V) → SAFE_OFF após 2s.
- RB-ELET-007: Religamento automático após sobrecarga com backoff 30s.
