# Regras de Dados

- RB-DADOS-001: Dados de telemetria retidos por 30 dias no SD.
- RB-DADOS-002: Dados de configuração retidos indefinidamente.
- RB-DADOS-003: Logs de segurança (audit trail) não rotativos.
- RB-DADOS-004: Export JSON de configuração inclui CRC32.
- RB-DADOS-005: Import valida schema + CRC antes de aplicar.
- RB-DADOS-006: Timeline de dados preserva timestamp UTC.
- RB-DADOS-007: Todos os dados persistidos têm owner module.
- RB-DADOS-008: Factory reset apaga dados de configuração.
- RB-DADOS-009: Dados de calibração mantidos pós-factory reset.
